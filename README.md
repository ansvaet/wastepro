## Установка

1. **Клонируйте репозиторий:**

   ```bash
   git clone https://github.com/ansvaet/wastepro.git
   cd wastepro
   ```

2. **Установите зависимости:**

   Убедитесь, что у вас установлен Node.js и npm. Затем выполните:

   ```bash
   npm install
   ```

## Запуск

1. **Запустите сервер разработки:**

   ```bash
   npm run dev
   ```

2. **Откройте браузер и перейдите по адресу:**
   
    Порт возможно задать через .env
   ```
   http://localhost:3000
   ```
3. **Билд проекта:**

   ```bash
   npm build
   npm run start
   ```


## Структура проекта

- `index.html` - Главная страница приложения.
- `js/analytics.js` - Скрипт для сбора и отправки аналитических данных.
- `js/app.js` - Основной скрипт приложения, включающий логику взаимодействия с пользователем.
- `js/form-handler.js` - Скрипт для обработки форм и валидации пользовательского ввода.
